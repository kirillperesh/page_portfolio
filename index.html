For now I've spent most of my time developing user authentication and product's CRUD and its testing.

    Main features:
  • LOTS of tests (divided into 4 groups: generic, forms, models and views)
  • Almost everything has docstrings and heavily (I suppose) commented
  • GitHub runs tests automatically on pull request
  • Custom user authentication pages
  • OAuth 2 authentication integration
  • taggit & photologue modules integration
  • Some nice photo upload/remove/reset features
